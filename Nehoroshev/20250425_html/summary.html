CSS

Для отображения элементов можно использовать таблицу стилей CSS. Таблица стилей состоит из селекторов и правил. Селекторы:

  • tag-name (имя тэга)
  • .className (имя класса)
  • #id (идентификатор, уникальное название)

table {
        border: 10px;
        border-color: black;
        width: 400px;
        height: 400px;
        border-style:groove;
    }
    td {
        border: 1px;
        border-style: solid;
        border-color: black;
    }
    .unknown {
        background-color: rgb(213, 164, 157);
    }
    .known {
        background-color: aquamarine;
    }

В примере выше: table, td - имена тэгов, unknown, known - имена классов

В фигурных скобках - правила. Пары "свойство-значение", разделённые двоеточием. Между правилами пишется точка с запятой.

Примеры правил: background-color - цвет фона. border - рамка. width, height - ширина и высота

Как добавить таблицу стилей?

  1. Создать файл appearance.css (или другое название)
  2. Написать в нём одно правило: 
            body {
            background-color: blue;
        }
  3. Добавить в свою html-страницу строчку в голову:
    4. <link rel="stylesheet" href="appearance.css">
